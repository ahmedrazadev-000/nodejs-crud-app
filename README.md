# Node.js CRUD Application

## Description
This is a simple CRUD (Create, Read, Update, Delete) application built using pure Node.js.

## Features
- Create new records using in-memory storage
- Read/view records
- Update existing records
- Delete records

## Technologies Used
- Node.js (no frameworks, no database)

## How to Run
1. Clone repository
2. Run `node app.js`
3. Open browser at `http://localhost:3000` (if using built-in HTTP server)
